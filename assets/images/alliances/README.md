# Alliance Logos

Drop official SVG logos here named by lowercase brand slug:
- `crowdstrike.svg`, `proofpoint.svg`, `imperva.svg`, `forescout.svg`,
  `cribl.svg`, `kriptos.svg`, `potech.svg`, `group-ib.svg`, `strangebee.svg`,
  `reveald.svg`, `automox.svg`, `tenable.svg`, `darkvore.svg`

Already present (CDN-fetched):
- `okta.svg`, `fortinet.svg`, `broadcom.svg`

When you add a new SVG, the alliance grid in `index.html` will pick it up
automatically via the `data-alliance-logo` flag (see `.alliance-card`).
